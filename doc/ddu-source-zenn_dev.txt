Introduction~
                                             *ddu-source-zenn_dev-introdution*

ddu.vim source collects articles in the repository for zenn.dev.


==============================================================================
Install~
                                                 *ddu-source-zenn_dev-install*

It depends on "zenn-cli", "|ddu|.vim", "|denops.vim|" and "|ddu-kind-file|".

https://zenn.dev/zenn/articles/zenn-cli-guide
https://github.com/Shougo/ddu.vim
https://github.com/vim-denops/denops.vim
https://github.com/Shougo/ddu-kind-file


==============================================================================
Examples~
                                                *ddu-source-zenn_dev-examples*

>
	" Use zenn.dev articles source.
	call ddu#start({
	    \     'sources': [{
	    \         'name': 'zenn_dev_article',
	    \         'options': {
	    \             'path': getcwd(),
	    \         }
	    \     }],
	    \     'kindOptions': {
	    \         'zenn_dev_article': { 'defaultAction': 'open' }
	    \     }
	    \ })
<


>
	" Search articles and browse it in zenn.dev.
	call ddu#start({
	    \     'sources': [{
	    \         'name': 'zenn_dev_article',
	    \         'params': {
	    \             'urlPrefix': 'https://zenn.dev/kyoh86',
	    \         }
	    \     }],
	    \     'kindOptions': {
	    \         'zenn_dev_article': { 'defaultAction': 'browse' }
	    \     }
	    \ })
<


==============================================================================
Params~
                                                  *ddu-source-zenn_dev-params*

urlPrefix
	(string)
	URL prefix to show the article

	Default: "http://localhost:8000"


slug
	(boolean)
	Whether to display the zenn slug (unique ID of the article) or not.

	Default: |v:true|


==============================================================================
Columns~
                                                 *ddu-source-zenn_dev-columns*

zenn_dev_emoji
	Shows Emoji of articles.

zenn_dev_slug
	Shows Slug of articles.

zenn_dev_title
	Shows Title of articles.


==============================================================================
Column Params~
                                           *ddu-source-zenn_dev-column-params*

limitLength
	(number)
	Max length of the column.

	Default: 0

==============================================================================
Action data~
                                             *ddu-source-zenn_dev-action-data*

Sources in this plugin are generating items with action data like below.

Article source:

	path
		File path for the article.

	date
		Date for the file staged on git.
	
	url
		The URL for the article under the localhost (preview).

	(Front matters)
		If you set some front-matter for the article, they will be
		expanded here  For example: >
		---
		title: "Le Petit Prince"
		emoji: "üêº"
		type: "tech"
		---
<
		they will beexpanded as fields.


==============================================================================
Actions~
                                                 *ddu-source-zenn_dev-actions*

This source uses |ddu-kind-file| actions.
You can see their actions in |ddu-kind-file-actions|.

And you can call "browse":
	Open the article in your browser.
	Parameters:
		opener
			An external command name to open it.
			e.g. "xdg-open", "wslview", etc.
